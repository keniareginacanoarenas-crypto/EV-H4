# EV-H4 – Evaluación H4

**Descripción:**  
Evaluación h4, kenia.com. Es una página base que podría funcionar como portafolio de diseño.

**Estructura del proyecto:**
- `src/` → archivos de código (HTML, CSS, JS)
- `assets/` → imágenes, íconos, fuentes
- `docs/` → documentación adicional
- `tests/` → pruebas si aplica

**Cómo usarlo:**  
1. Abrir `index.html` en un navegador.
2. Revisar estilos en `style.css`.
3. Modificar scripts en `script.js`.
